# N＋とは／N＋というのは

例句: 人生とは本人が主役のドラマみたいなものだ。
意思: 下定义；解释说明

翻译: 所谓人生，就像是一部由自己担任主角的连续剧。
说明: 「と」引用，「の」名词化。常用句型：～とは～のことです／～とは～の略です。

`N+とは/というのは` ****

**`と`引用的内容；`の`名词化。Aという（叫做 A） ，A という + の（叫做 A 的这件事/这个词）** 

（1）“所谓……”、“……是指”。**下定义** 

下定义常见格式：`～とは、～のことです`（所谓A，就是指B。）

缩写说明格式：`～とは、～の略（りゃく）です。`（A是B的缩写。）

- **人生（じんせい）とは**本人（ほんにん）が主役（しゅやく）のドラマみたいなもの**だ**。
    
    所谓人生，就像是一部由自己担任主角的连续剧。
    

（2）**解释说明；阐述原因/理由，未必是明确的因果关系。**

- 今、ふところがさびしいんだ。というのは、まだ給料（きゅうりょう）もらってないんだから。我现在手头有点紧。（原因）是因为还没领到工资呢。
