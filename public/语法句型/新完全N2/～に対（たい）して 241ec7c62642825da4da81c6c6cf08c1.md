# ～に対（たい）して

例句: 目上の人に対してそんな乱暴な言い方をしてはいけません。
意思: “对于，针对”；“与之相对的”
日期: April 3, 2026 3:47 PM
翻译: 对长辈不可以用那么粗鲁的说法。
说明: 表示动作、态度、评价等指向的对象；也可表示对比（Aに対してB）。

**`〜に対して`　对す（たいす）**

（1）表示动作、态度的“对象”。**“对于，针对”**

接续形式：N + に対して；N + に対する + 名词 (修饰名词时)

- 目上の人（めうえのひと）に対してそんな乱暴（らんぼう）な言い方をしてはいけません。
    
    不可以对长辈（或上级）用那样粗鲁的方式说话。
    

（2）表示明确的“对比/对照”。**“与之……相对”**

接续：N+に対して；～ + の + に対して

- 朝（あさ）はパンを食（た）べると答えた人が30%だったのに対して、ご飯を食べると答えた人は65%だった。回答早上吃面包的人占 30%，与之相对回答吃米饭的人则占了 65%。
