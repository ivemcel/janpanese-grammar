# 小句（简体）+ N

意思: 名词修饰 (定语从句)
例句: これは 明日会議で使う 資料です
わたしが明日乗る 飛行機は中国航空です
中国で買った CDを友達に貸しました
入口の広い 建物；操作が簡単な パソコン
翻译: 这是明天会议要用的资料
我明天乘坐的飞机是中国航空公司的
我把在中国买的CD借给朋友了
分类: 名词修饰
