# N+でも/疑问词+でも/～でも～でも

意思: 举出同类事物中之一为例
例句: 映画でも見に行きませんか。
翻译: 要不要去看个电影之类的？

`N+でも` 列举

- 映画（えいが）でも見（み）に行（い）きませんか。（第37课）要不要去看个电影之类的
- ~~わたしは昨日喫茶店でコーヒーでも飲みました。~~一般不用于过去式句子。

`疑问词+でも` 无论…都…

- 誰（だれ）でも 参加（さんか）できます。（第17课）所有人都可以参加

`て形+も`：表逆接，“即使”，提示极端的例子。（参考第35课）

- 雨でも、行きます。（**即使**下雨，**也**要去。）

`～でも～でも` ****N1 + でも + N2 + でも。“不管是A还是B”、“无论A还是B”。

- 彼が主演する映画は、アクションでも恋愛ドラマでも、必ず当たります。 他主演的电影，不管是动作片还是爱情剧，都一定会大火（大卖）。
- お年寄りであっても、子供であっても、～。不管是老人，还是小孩，……
