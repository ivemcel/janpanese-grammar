# ～までに

意思: までに在...之前 (截止期限);まで（直到）
例句: 明日の９時までに この書類を完成させなければなりません。
翻译: 明天9点之前，必须完成这份文件。
分类: 时间/顺序

**まで（直到）** 

**までに（在……之前）**

| **语法** | **含义** | **例句** |
| --- | --- | --- |
| **～まで** | 动作或状态的**持续**。 | 明日の9時**まで**働きます。(一直工作**到**9点) |
| **～までに** | 在某个节点前完成（一次性/完结性） | 明日の9時**までに**終わります。(在9点**前**做完) |
| **～間** | 动作持续的期间（状态） | 学校が休みの間 アルバイトをする。**整个假期**我都在打工。 |
| **～間に** | 动作发生的时间范围（一次性/完结性） | 学校が休みの間に アルバイトをする。（可能假期有两个月，我只打了三天工） |
