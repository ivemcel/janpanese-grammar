`～ではないか/～ではないだろうか/～ではないでしょうか` ”难道不是……吗？”“我认为是……”

接续：**A/V（简）+ の + ではないか；AN / N + ではないか**

在日常口语中：

- の ＝ ん  *のではないか* → *んじゃないか*（口语最常用）。
- では ＝ じゃ *ではない* → *じゃない*。
- ないです ＝ ありません *～ではないでしょうか* 也可以说成 *～ではありませんか*。
