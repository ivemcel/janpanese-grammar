せっかく + の + N  难得的～ / 宝贵的～

- せっかくのチャンスだから、（あきらめないで頑張りたいと思います。）
    
    因为是难得的机会，（我想不放弃，努力试试看。）
    

**せっかく + だから（接受）：** 既然好不容易……就……

**せっかく + だけど / ですが（婉拒）：** 虽然难得……但是……

- せっかくだから、遠慮なくいただくよ。既然你这么热心，那我就不客气地收下/吃了。
- せっかくのお誘いですが、友達との約束がありまして……承蒙您特意邀请，但我已经和朋友有约了……。

せっかく + ～のに / ＋な＋のに 好不容易……竟然……（表达遗憾、可惜或埋怨）

- せっかく来たのに。好不容易来了一趟，结果却……
