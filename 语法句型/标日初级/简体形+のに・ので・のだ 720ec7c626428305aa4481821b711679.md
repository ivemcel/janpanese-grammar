# 简体形+のに・ので・のだ

意思: のに明明...却... (事实性逆接)，表示惊讶、抱怨等语气。
例句: 明明设了闹钟，今早却没能起来。
翻译: 目覚ましをかけておいたのに、今朝は起きられませんでした。

**`简体形+のに・ので・のだ`** 

～のに：事实性逆接，“明明～”。表示惊讶、抱怨等语气。AN/Nな＋のに

- 目覚ましをかけておいたのに、今朝は起きられませんでした。明明设了闹钟，今早却没能起来。
- 雨が降っているのに、出かけます。明明在下雨，却还要出门。
- 今までにゴルフをしたことがないんですって。**初めてな** のに上手ですね。
- **せっかく**の日曜日な**のに**、雨が降った。（难得 的周日，却下雨了。）
- **せっかく**来てくれた**のに**、留守にしてすみませんでした。您**特意**跑一趟来看我，我**却**不在家，真是对不起。

～ので ：表示原因、理由（因为……所以……）。

- 雨が降っているので、出かけません。

～のだ / ～んだ :表示强调、解释说明。

- 外は雨が降っているのです（んだ）。是因为外面在下雨呀。

**注意点：**

- 双节棍句（前后是独立的句子），ので、から、が、けど前面可以接简体或者敬体
- ～のに 后项一般为事实，所以不接命令、依赖、意志、判断、疑问等表达。
    - X 寒いのに、換気のために窓を開けてください。（命令）
    - X 日曜日なのに、森さんは会社にいるだろう。（推量）
    - X 日曜日なのに、学校に行きますか。（疑问）
- のに可用于句末
    - せっかく来たのに。特意过来了
